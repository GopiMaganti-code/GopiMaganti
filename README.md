## 👋 About Me

I am an **Automation Test Engineer (SDET)** focused on **high-signal test automation** - from stable UI flows to contract-aware API checks. I work primarily with **Playwright** and **JavaScript/Node.js**, and I validate services with **Postman** and **Newman** for collections, environments, and CI-ready runs.

I am strongest where **UI and API layers meet**: tracing failures across the stack, **debugging flaky behavior**, and turning noisy logs into **clear, actionable defects**. I care about **end-to-end coverage** that reflects real user journeys while keeping suites **fast, parallel-friendly, and easy to extend**.

## ✨ What Makes Me Different

- **Production-shaped testing** - scenarios that mirror real usage, data, and timing, not only happy paths.
- **Flake resistance** - smart waits, isolation, diagnostics, and cleanup so failures point to **product or env issues**, not the framework.
- **Stability as a habit** - consistent patterns, clear ownership of test data, and reporting that helps teams **fix root causes**, not re-run blindly.
